此插件是為了偷懶和方便分辨/觀看人物之間的對話以及旁白的團錄轉換器。

例如Discord跑圖記錄，把輸出的txt檔案扔進去，會發現人可以懶惰，但不多。

圖片URL不建議使用DC的URL（會有失聯風險）
*推薦Plurk的圖片URL

所有東西都可以透過修改自己的txt來讓生成內容變得正常，所以還是不可以那麼懶。
